⚛ NOÜMIND ELECTRON LAUNCHER - ФАЙЛЫ И ОПИСАНИЕ

Полный проект с 10+ файлами и ~50KB кода + 180MB Electron

ОСНОВНЫЕ ФАЙЛЫ (Рабочие):
───────────────────────────────────────────────────────────────────────────
✓ main.js (5.5 KB)
  - Electron main process
  - Управление окном, IPC, Python процесс
  - Запуск/остановка узла

✓ renderer.js (8.8 KB)
  - UI логика (чат, метрики, toggle)
  - HTTP запросы на Gateway
  - Слушание IPC событий

✓ preload.js (1.1 KB)
  - IPC безопасность (contextBridge)
  - API для renderer процесса

✓ index.html (12 KB)
  - UI структура (HTML)
  - CSS стили (тёмная тема, оранжевые акценты)
  - Два столбца: 60% чат + 40% узел

✓ config.js (3.1 KB)
  - Центральная конфигурация
  - Gateway URL, параметры узла, UI интервалы
  - Feature flags, EFCT параметры

✓ package.json (311 bytes)
  - npm конфигурация
  - Зависимости (только Electron)
  - Скрипты запуска

ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:
===============================================================================

1️⃣ УСТАНОВКА:
   $ npm install
   $ ./scripts/check-requirements.sh
   $ npm start

2️⃣ ЗАПУСК:
   $ npm start
   → Открывается окно Electron
   → Python узел загружается
   → Можно писать сообщения

3️⃣ API ЗАПРОС (из другого приложения):
   $ curl -X POST http://localhost:8000/chat \
     -d '{"message": "Hi"}'

4️⃣ РАЗРАБОТКА:
   $ npm start
   → Cmd+Option+I для DevTools
   → Редактируйте файлы
   → Обновление происходит автоматически (или рестарт)
