# 🚀 Первый запуск Noümind Launcher

**Всё максимально просто!** Только 2 команды:

## ⚡ Быстрый старт

```bash
cd ~/noumind-launcher
npm start
```

**Вот и всё!** ✨

## 🎯 Что происходит при `npm start`:

1. **Проверка SSH** - приложение проверяет доступ к VPS (87.106.255.55)

2. **Автоматическая загрузка файлов** (если их нет):
   - `node_gguf.py` (~15KB) - ⚡ быстро
   - `gguf_inference_fallback.py` (~8KB) - ⚡ быстро
   - `efct_com.py` (~11KB) - ⚡ быстро
   - `model.gguf` (600MB) - ⏳ может быть долгим (зависит от интернета)

3. **Запуск Electron приложения** - откроется окно

4. **Автоматический запуск узла** - если файлы загружены

5. **Готово к использованию** - пишите сообщения!

## 📊 Примерное время

| Операция | Время |
|----------|-------|
| Проверка SSH | 3 сек |
| Скачивание .py файлов | 10 сек |
| Скачивание модели (600MB) | 1-5 минут (зависит от интернета) |
| Запуск приложения | 2 сек |
| Загрузка модели в памяти | 15 сек |
| **ИТОГО** | **2-7 минут** |

## 💡 Советы

### Интернет медленный?
Модель (600MB) скачивается в фоне. Пока она загружается:
- Приложение уже открыто
- Вы можете использовать Gateway (удаленные узлы)
- Когда модель загрузится - автоматически запустится локальный узел

### Нет SSH доступа?
Если нет доступа к VPS через SSH:
1. Приложение скажет об этом в консоли
2. Вы сможете загрузить файлы вручную (см. INSTALL.md)
3. Приложение будет работать через Gateway (удаленные узлы)

### Что-то пошло не так?
Смотрите INSTALL.md раздел Troubleshooting

## 🆘 Проблемы при первом запуске?

### Ошибка: "npm: command not found"
→ Установите Node.js с https://nodejs.org

### Ошибка: "scp: command not found"
→ SSH не установлен
- macOS: `brew install openssh`
- Linux: `sudo apt install openssh-client`
- Windows: Используйте WSL или установите OpenSSH

### Ошибка: "Permission denied"
→ Проверьте что у вас есть доступ к VPS (SSH ключ настроен)

### Модель загружается очень долго
→ Это нормально (600MB), оставьте приложение открытым

### Нет интернета / нет доступа к VPS
→ Приложение предложит использовать Gateway (удаленные узлы)
