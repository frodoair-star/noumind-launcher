# 🎯 Начните отсюда - Noümind Launcher

**Это очень просто!** Только одна команда:

```bash
npm start
```

Всё остальное происходит **автоматически**! ✨

---

## ⚡ За 2 минуты:

```bash
# 1. Перейдите в папку проекта
cd ~/noumind-launcher

# 2. Запустите приложение
npm start

# Готово! Пишите в чат 💬
```

---

## 🤖 Что происходит автоматически:

✅ **Проверка доступа к VPS** - 3 сек  
✅ **Скачивание файлов** (если нужно) - 1-5 минут  
✅ **Запуск приложения** - 2 сек  
✅ **Загрузка модели** - 15 сек  
✅ **Готово к использованию!** 🎉  

**Никаких команд в терминале не нужно вводить!**

---

## 📊 Первое сообщение

1. Дождитесь пока приложение полностью загрузится (прогресс-бар 100%)
2. Напишите вопрос в чат: `"What is AI?"`
3. Нажмите Enter
4. Получите ответ за 8-15 секунд

---

## 🆘 Если что-то не работает

| Проблема | Решение |
|----------|---------|
| "npm: command not found" | Установите Node.js: https://nodejs.org |
| "scp: command not found" | SSH не установлен (но это редко) |
| Очень медленная загрузка | Модель (600MB) скачивается, это нормально |
| Нет доступа к VPS | Приложение будет работать через Gateway (удаленные узлы) |

Больше помощи: смотрите **FIRST-RUN.md** или **INSTALL.md**
